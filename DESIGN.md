# Design System Specification
 
## Visual Direction
This design system transition moves to a clear, high-visibility **light mode** environment. The aesthetic is professional, modern, and highly legible, utilizing a balanced color palette and standard geometric proportions to ensure a versatile user experience across various interface types.
 
## Color Palette
The system utilizes a fidelity-based color logic that emphasizes brand-accurate representation:
- **Primary Color**: #3B82F6 (Vibrant Blue) - Used for primary actions, active states, and key brand highlights.
- **Secondary Color**: #10B981 (Emerald Green) - Used for supporting information, success states, and secondary UI elements.
- **Tertiary Color**: #EF4444 (Soft Red) - Reserved for accents, error states, or specific semantic highlights.
- **Neutral Base**: Focused on light surfaces with high-contrast text for maximum readability.
 
## Typography
The typographic scale is designed for clarity and a contemporary feel:
- **Headlines**: Manrope – A geometric sans-serif that provides a clean, modern structure to page headers.
- **Body & Labels**: Inter – Selected for its exceptional legibility at small sizes and high-density information displays.
 
## Shape & Structure
The system employs a standardized geometric approach to spatial relationships:
- **Roundedness**: Moderate (2) – All components feature softened corners that provide a friendly yet professional appearance, avoiding the harshness of sharp angles while remaining structured.
- **Spacing**: Normal (2) – A standard density scale that provides sufficient breathing room between elements without sacrificing information density.
 
## Component Principles
- **Clarity**: Components are defined by clear boundaries and high contrast against the light background.
- **Responsiveness**: The combination of Inter and Manrope ensures that hierarchy is maintained across mobile and desktop viewports.
- **Affordance**: Interactive elements use the primary blue to signify clickability clearly within the light interface.
